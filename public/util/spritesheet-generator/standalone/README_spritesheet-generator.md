# Spritesheet Generator

Kurzanleitung

1. Öffne die Datei `spritesheet-generator.html` in einem Browser (Chrome, Firefox, Edge).
2. Wähle mehrere PNG-Frames (z. B. `0001.png`, `0002.png`, ...) per Drag & Drop oder über den Dateiauswahl-Dialog.
3. (Optional) Aktiviere "Automatisch numerisch sortieren" oder ändere "Frames pro Zeile" bzw. "Abstand px".
4. Klick auf "Spritesheet erstellen". Das Ergebnis wird als PNG im Canvas angezeigt.
5. Klick auf "PNG herunterladen", um die Datei `spritesheet.png` zu speichern und in Moodle hochzuladen.
